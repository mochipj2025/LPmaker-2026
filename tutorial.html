📖 使い方ガイド

LP StructureMaker ― はじめてでも、選んで作れる。LPの「構成の設計図」から画像まで揃います。 無料 / 60秒

✨これは何のツール？

LP構成案を作る最短ルートと、出力タイプ・詳細設定の使い方をまとめています。 基本だけ知りたい場合はツール本体からそのまま始められます。

🔧 もっと詳しく（最短の道・出力タイプ・こだわる道・業種早見表） ▼

⚡ 最短の道：3ステップ（約60秒）

迷ったらこれだけ。①左上のプリセットを1つクリック（業種・読み手・ゴール・トーンが一瞬で埋まり、設計図が自動でコピーされます）→ ②右の見取り図で形を確認（ブロックをクリックすると該当セクションの書き方がハイライト）→ ③ChatGPTかClaudeに貼って送信。

📤 出力タイプ：3つの使い方

基本設定のいちばん下「出力タイプ」で切り替えます。 🧱 構成の設計図＝セクションの順番と書くことのガイド（まず考えたい人向け）。 🚀 LP制作プロンプト＝AIにLPそのものを作らせる指示文（そのまま作りたい人向け。Claude / ChatGPT / v0 に貼るとその場でプレビューが出るよう指示済み）。 🖼️ 必要な画像プロンプト集＝このLPに必要な画像1枚ずつの生成プロンプト（画像を用意する人向け）。 切り替えても選択内容は引き継がれます。

🔧 こだわる道：作り込みたい人へ

掲載情報（店名・価格・日時など）を入れると、設計図に「正確に載せる」指定が入り、 空欄でも「架空の実績を作らない」安全指示が自動で入ります。 読み手を選び直すと、共感・実績・料金・お知らせなど強調するセクションが変わります。 画像は姉妹ツール Visual PromptMaker の Graphic タブで、設計図に書かれたテイストのまま作ると世界観が揃います。

🏪 業種で構成はこう変わる（早見表）

「悩み解決」の型を全業種に当てはめない、がこのツールの考え方です。

業種構成の主役
飲食店・カフェ・バーコンセプト → 看板メニュー → 雰囲気
サロン・整体・クリニック共感（悩み）→ 選ばれる理由 → 実績
SaaS・アプリ課題 → 機能 → 料金プラン
EC・物販ベネフィット → こだわり → レビュー
セミナー・講座得られる未来 → カリキュラム → 講師
コーポレート・個人提案の核 → 強み → 実績
