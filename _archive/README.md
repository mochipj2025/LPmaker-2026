# _archive ― 退避した旧実装（失敗ログ）

このフォルダは、現行の入口設計に置き換えられた**旧バージョンを「失敗ログ」として保管**する場所。
消さずに残すのは、同じ設計ミスを繰り返さないため／後から判断の経緯を追えるようにするため。

## wizard/（旧ステップ版）

- **退避日**: 2026-07-17
- **役割だったもの**: `index.html`（全項目1画面）を1問ずつ進める最初のステップ版。
- **退避理由（失敗ログ）**:
  - 新しい情報設計でやり直した `v2/` に置き換えられた。
  - 2026-07 に `index.html` のヘッダーリンクが v2 へ差し替わって以降、**どこからもリンクされない孤児**になっていた（到達不能なまま残存）。
  - ステップ版としては v2 に一本化する方針のため、表からは退場。
- **温存する価値**: `wizard/js/wizard.js` は「app.js の上に薄く被せてステップ進行を足す」実装で、
  v2 が画像作成・アイコン作成のステップ対応をするときの**参考コード**になりうる。
- **動作**: 相対パス（`<base href="../">`）で親の `js/` を読む作りだったため、この階層に移した状態では
  そのままでは動かない。あくまで参照用。
