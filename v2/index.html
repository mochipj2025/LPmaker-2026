    公開したくなったときに

    サイトとして公開する準備（あとでOK）

    これはあとで自分のサイトとして公開したくなったときだけの準備です。設計図やAIへの指示文を作るだけなら要りません。
    やるなら、下のボタンでコピーして Claude Code や ChatGPT に貼るだけ（中身は読まなくてOK）。

    中身を見る（読まなくてもOK） 
    新しいプロジェクト用フォルダ「(サイト名をここに)」を作成し、その中に移動して git init を実行してください。
続けて GitHub に同名の空リポジトリを作成してください
（gh コマンドが使えるなら `gh repo create (サイト名) --public --source=. --remote=origin --push` を実行、
使えない場合は https://github.com/new から手動で作成してもらい、できたリポジトリのURLを私に教えてください）。
リポジトリができたら、このフォルダをそのリモート（origin）にひも付けて、空コミットを1つ push してください。
終わったら「フォルダとリポジトリの準備ができました」と教えてください。
